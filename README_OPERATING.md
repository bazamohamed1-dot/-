# دليل التشغيل (Operating Guide)

هذا الدليل يشرح كيفية تشغيل التطبيق بشكل صحيح لتجنب المشاكل الشائعة.

## الترتيب الصحيح للتشغيل

يجب اتباع هذا الترتيب بدقة:

1.  **تشغيل Docker Desktop:**
    *   تأكد من أن برنامج Docker يعمل وأن الأيقونة (الحوت) ظاهرة بجوار الساعة.
    *   انتظر حتى يظهر الشريط الأخضر (Engine Running).

2.  **إعداد النفق (مرة واحدة فقط):**
    *   قم بتشغيل ملف `setup_tunnel.bat` **مرة واحدة فقط** عند تثبيت المشروع لأول مرة.
    *   هذا الملف يقوم بنسخ ملفات الاعتماد (Credentials) إلى المكان الصحيح. لا تحتاج لتشغيله كل مرة.

3.  **تشغيل التطبيق:**
    *   اضغط مرتين على ملف `run_docker.bat`.
    *   ستظهر شاشة سوداء تقوم ببناء النظام وتشغيل الخدمات.
    *   **هام:** لا تغلق هذه الشاشة السوداء. إذا أغلقتها، سيتوقف عرض السجلات (Logs)، لكن السيرفر قد يظل يعمل في الخلفية.

## حل المشاكل الشائعة

### 1. الشاشة السوداء تظهر وتختفي فوراً
هذا يعني وجود خطأ يمنع السكربت من البدء. لقد قمنا بتحديث الملف ليبقى مفتوحاً، لكن إذا استمرت المشكلة:
*   تأكد أن Docker Desktop يعمل.
*   تأكد أنك قمت بفك الضغط عن المشروع بشكل صحيح.

### 2. الدائرة الخضراء (Running) لكن الموقع لا يعمل
الدائرة الخضراء تعني فقط أن "الحاوية" تعمل، لكن قد يكون البرنامج بداخلها يواجه خطأ (مثل خطأ في الكود أو قاعدة البيانات).
*   **الحل:** انظر إلى الشاشة السوداء (Logs). ابحث عن كلمات مثل `Error` أو `Traceback`.
*   تأكد من تحديث الكود لآخر نسخة (لقد قمنا بإصلاح مكتبات ناقصة مؤخراً).

### 3. الدخول للنظام
*   **محلياً:** [http://localhost:8000](http://localhost:8000)
*   **عبر الإنترنت:** [https://admin.bazasystems.com](https://admin.bazasystems.com) (يتطلب أن يكون النفق Cloudflared متصلاً بنجاح).

## ملاحظة حول اسم المجلد
لقد تم ضبط الإعدادات لتعمل مع اسم المجلد `baza-app`. لا تقم بتغيير اسم المجلد مرة أخرى لتجنب تضارب الإعدادات.
